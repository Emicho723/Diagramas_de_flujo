Descripción

Este trabajo muestra, mediante un diagrama de flujo, el funcionamiento lógico del programa correspondiente al Deber 1.

El programa se encarga de realizar una conversión entre diferentes divisas. Primero, el usuario debe ingresar una moneda base y otra moneda de referencia. Luego, el sistema verifica que los códigos ingresados estén formados por tres letras. Después de esta validación, el usuario introduce la tasa de cambio junto con el monto inicial.

A partir de estos datos, el programa calcula el valor convertido y presenta un reporte denominado Market Snapshot. Finalmente, el usuario tiene la opción de realizar una nueva conversión o finalizar la ejecución del programa.

En este repositorio se encuentran tanto el código del programa como el diagrama de flujo que describe la lógica utilizada en el algoritmo.
